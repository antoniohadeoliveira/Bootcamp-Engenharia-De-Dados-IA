Meu objetivo principal de carreira é me tornar Engenheiro de Dados. E este bootcamp será o meu pontapé inicial. O objetivo é aprender todos os ensinamentos passados pelo Professor Demetrius, e com esse conhecimento adquirido, inserir-me no mercado de engenharia de dados como um engenheiro júnior ou até mesmo como trainee. Também tenho como objetivo no bootcamp começar a formar um fundamento sólido de conhecimento e uma stack para começar a trabalhar nesta área.
